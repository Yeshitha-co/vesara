# VESARA - Premium Spices & Dry Fruits Wholesale Website

A modern, SEO-optimized B2B e-commerce website for Y S Creations - VESARA, built with Next.js 16, Tailwind CSS, and deployed on Vercel.

## ✨ Features

✅ **Modern Design** - Beautiful gold and maroon branding
✅ **SEO Optimized** - Meta tags, Open Graph, sitemap, robots.txt
✅ **Mobile Responsive** - Perfect on all devices  
✅ **Fast Performance** - Vercel CDN
✅ **B2B Focused** - Products, quotations, contact
✅ **Zero Cost** - Free Vercel hosting
✅ **Easy Updates** - Simple file structure

## 🚀 Deploy in 5 Minutes

### Step 1: Local Setup
```bash
npm install
npm run dev
# Open http://localhost:3000
```

### Step 2: Push to GitHub
```bash
git init
git add .
git commit -m "VESARA website"
# Push to your GitHub repo
```

### Step 3: Deploy to Vercel
1. Go to [vercel.com](https://vercel.com)
2. Click "Import Project"
3. Connect your GitHub repo
4. Click "Deploy"
5. **LIVE!** 🎉
